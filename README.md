## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), a limited sandbox for trying out app development with Expo

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).

# Mobile Weather Application 🌤️📱

A modern, responsive, and lightweight cross-platform mobile weather application built using **React Native** and **Expo**. This app provides real-time weather updates, forecasts, and interactive UI animations for both **Android** and **iOS** devices from a single codebase.

---

## 📌 Project Overview

The main objective of this project is to deliver an intuitive and visually appealing weather tracking experience on mobile devices. By leveraging React Native's cross-platform capabilities, the app ensures native performance and consistent UI/UX across different operating systems.

## ✨ Core Features

* 🌍 **Real-time Weather Data:** Fetches live weather conditions (Temperature, Humidity, Wind Speed, UV Index) based on the user's current location.
* 🔍 **City Search:** Allows users to search for weather updates of any city worldwide.
* 📅 **7-Day Weather Forecast:** Provides accurate daily and hourly weather forecasts.
* 🎨 **Dynamic UI Backgrounds:** Automatically changes background themes and icons based on the current weather condition (e.g., Sunny, Rainy, Snowy, Stormy).
* 📍 **GPS Location Integration:** Automatically detects location permissions to show instant local weather.
* 📱 **Cross-Platform Optimization:** Smooth 60fps animations and responsive layouts tailored for both iOS and Android.

---

## 🛠️ Tech Stack & Libraries

* **Framework:** React Native (Expo Workflow)
* **Language:** JavaScript / TypeScript
* **Styling:** Tailwind CSS (NativeWind) / React Native StyleSheet
* **API Integration:** OpenWeatherMap API (or WeatherAPI)
* **Icons:** React Native Vector Icons / Lucide React Native
* **Location Services:** `expo-location`

## 🚀 Getting Started

Follow these steps to set up and run the project locally:

### 1. Prerequisites
Make sure you have Node.js installed on your machine. You will also need the Expo Go app on your physical device to test it.

### 2. Installation
Clone the repository and install the dependencies:

```bash
# Clone the repository
git clone [https://github.com/savindunawanjana2003/Mobail-app-react-native.git](https://github.com/savindunawanjana2003/Mobail-app-react-native.git)

# Navigate into the project directory
cd Mobail-app-react-native

# Install dependencies
npm install
