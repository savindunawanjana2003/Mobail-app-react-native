# 🌤️ Weather App with Gemini AI Chatbot

A powerful, modern cross-platform mobile application built with **React Native** and **Expo**. This app provides real-time weather forecasts based on the user's current location, allows saving favorite locations, and features an advanced AI Chatbot powered by Google's **Gemini API** to answer any weather-related questions.

---

## 🚀 Features

*   **📍 Real-time Location Weather:** Automatically detects your current location to show accurate, up-to-date weather details and a 5-day forecast.
*   **💾 Location Management (Save & View):** Save your favorite cities or locations and easily switch between them to check their specific weather details instantly.
*   **🤖 Integrated Gemini AI Chatbot:** Connected with Google's Gemini API, allowing users to chat and ask complex weather-related questions, tips, or climate insights.
*   **📱 Cross-Platform Experience:** Smooth performance and consistent UI across both Android and iOS devices.

---

## 🛠️ Tech Stack & Services Used

*   **Framework:** [React Native](https://reactnative.dev/) (Cross-platform Mobile Development)
*   **Tooling & Workflow:** [Expo](https://expo.dev/) (Managed workflow with Expo Router & Expo EAS Build)
*   **AI Integration:** [Google Gemini API](https://ai.google.dev/) (For the smart weather assistant)
*   **State Management & Navigation:** Expo Router (File-based routing)
*   **Styling:** NativeWind / Tailwind CSS for fully responsive and clean UI designs.
